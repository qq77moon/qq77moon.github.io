获取Let's Encrypt免费SSL证书

发表于 2018-08-24 | 分类于 运维

1.SSL证书

1.1 获取证书

获取Let’s Encrypt免费SSL证书，首先获取脚本:

git clone https://github.com/letsencrypt/letsencrypt
cd letsencrypt
chmod a+x certbot-auto

使用Certbot客户端申请证书(注意，有可能ngixn启动导致80端口被占用无法获取，建议先停止nginx):

./certbot-auto certonly  -d "*.shandudata.com" --manual --preferred-challenges dns-01  --server https://acme-v02.api.letsencrypt.org/directory

3.2 nginx配置证书

在nginx配置文件中添加对上面生成的文件的引用就能使用ssl证书，如下是例子:

server {
    listen 443 ssl;
    server_name jumpserver.shandudata.com;
    ssl on;
    ssl_certificate /etc/letsencrypt/live/shdandudata.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/shdandudata.com/privkey.pem;
    ssl_trusted_certificate  /etc/letsencrypt/live/shdandudata.com/chain.pem;
    ssl_session_timeout 5m;
    ssl_protocols TLSv1 TLSv1.1 TLSv1.2;
    ssl_ciphers AESGCM:ALL:!DH:!EXPORT:!RC4:+HIGH:!MEDIUM:!LOW:!aNULL:!eNULL;
    ssl_prefer_server_ciphers   on;

    ……
}

3.3 证书续期

Let’s encrypt 的免费证书默认有效期为 90 天，可以添加crontab自动获取最新证书:

0 */12 * * * /home/deploy/github/letsencrypt/certbot-auto renew --renew-hook "/etc/init.d/nginx reload"
