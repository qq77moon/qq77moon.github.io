Jumpserver使用手册

发表于 2018-08-27 | 分类于 运维

1. 前言

为了规范主机登录，管理主机操作，后续的服务器操作请全部通过jumpserver执行。所有的主机操作命令都会后台记录，请大家一定注意命令安全。

2. Web登录

jumpserver服务web地址为: https://jumpserver.shandudata.com/ ，登录用户名为个人姓名拼音，密码之前已经通过个人邮箱发送过去，如果忘记密码或者找不到邮件请联系我发送重制密码邮件。

登录页面：

进入登录页面后，左侧为操作菜单，右侧为个人能够登录的主机资源，如果发现自己需要登录的主机不在资产列表中，请联系我添加权限。

个人信息页面能够修改个人基本信息、个人密码、ssh密钥和mfa设置：

web终端菜单会进入服务器登录界面，左侧为个人能够登录的资产列表，点击需要登录的主机，右侧会正常登录主机，然后可以和term终端一样操作主机了。

3. ssh登录

如果觉得登录web页面麻烦，也可以直接在自己的term终端中使用ssh登录主机。操作命令如下：

ssh -p2222 fengqiqi@jumpserver.shandata.com

或者

ssh -p2222 fengqiqi@45.40.248.73

请注意，将上面的fenqiqi换成你自己的jumpserver个人账号，然后会提示输入密码，密码也是jumpserver用户的密码。

成功进入跳板机界面后按照上面提示输入操作命令，一般可以先输入p查看自己能够登录的资产列表，然后输入需要登录主机的id或ip，如果由多个用户的主机还会提示选择用户。成功登录主机后就可以正常操作主机了。

4. sftp传输

如果有文件传输的需求，可以通过在term终端中通过sftp命令执行，命令如下：

sftp -P2222 fengqiqi@jumpserver.shandudata.com

或者

sftp -P2222 fengqiqi@45.40.248.73

请注意，上面的P是大写的，将上面的fenqiqi换成你自己的jumpserver个人账号，然后会提示输入密码，密码也是jumpserver用户的密码。

成功进入跳板机后用ls可以显示主机列表，依次进入主机名、用户名，就成功进入主机目录。

请注意，由于sftp无法切换应用用户，为了解决能登录所有主机的问题，采用的是jumpserver普通用户去登录主机，进入的主机目录是/tmp目录（默认所有主机的tmp目录是全权限的），其他目录没有权限操作，get和put文件都是在/tmp目录下进行的，所以如果需要下载文件，请先将文件拷贝到tmp目录，上传文件上传后请从tmp目录下拷贝到自己想要的目录。

5. fabric批量执行

为了能够简化一些重复操作，额外提供了fabric脚本支持，执行fabric脚本会通过jumpserver服务器跳转到目标主机执行。下面是一个在目标主机新建一个fff.txt文件的fabric脚本案例（下面fabric采用的是fabric2.3）：

# -*- coding: utf-8 -*-
#

from fabric import task, Connection

conn = Connection(host='deploy@10.6.0.170', forward_agent=True, connect_timeout=10000,
              connect_kwargs={'password': 'sddata-deploy'},
              gateway=Connection('fengqiqi@45.40.248.73:3333', connect_kwargs={'password': 'XXXXXXXX'},
                                 connect_timeout=10000))

@task
def touchfile(c):
    conn.run('touch fff.txt')

需要实现fabric通过jumpserver只需要注意以下几点：

1、在Connection中添加gateway选项，具体如下：

gateway=Connection(‘fengqiqi@45.40.248.73:3333’, connect_kwargs={‘password’: ‘xxxxxxx’})

请将上面的fengqiqi更换城自己的jumpserver用户名，password后面为jumpserver密码。

2、host为需要执行操作的目标主机ip，如果该主机没有外网ip，请换成该主机的openvpn的ip，如果该主机没有搭建openvpn请联系我。

然后按照正常的fabric脚本执行即可：
